Tipos de Listas

Ordenadas

Estas são listas que seguem um parâmetro de ordenação como por exemplo sequência numérica ou do alfabeto. Exemplos a seguir:

  1. Azul
  2. Amarelo
  3. Verde
  4. Laranja

O exemplo acima seguiu a sequência numérica, agora outro exemplo seguindo a sequência alfabética

  1. Janeiro
  2. Fevereiro
  3. Março
  4. Abril

Não ordenadas

Este tipo de lista não é marcada por uma sequência numérica ou alfabética, apenas por símbolos e/ou desenhos

  • Arroz
  • Feijão
  • Macarrão
  • Oleo

Listas de Definição

Estão são listas para atribuir significados, descrições, etc. Vejamos os exemplos a seguir:

HTML
Linguagem de Marcação
CSS
Estilização do site
JavaScript
Interação, essa sim é uma linguagem de programação
